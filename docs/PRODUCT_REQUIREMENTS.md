# TopiIsland macOS 灵动岛需求文档

## 产品定位

TopiIsland 是一款 macOS 原生灵动岛应用。它常驻屏幕顶部正中间，以胶囊形态呈现系统状态、媒体播放、快捷操作、任务进度、通知、剪贴板、日历待办和 AI 快捷入口。

产品目标不是做一个普通浮窗，而是做一个系统级轻量交互入口：视觉精致、动画优雅、交互克制、性能稳定、隐私可控。

## 平台与技术

- 平台：macOS 14+
- 语言：Swift
- UI：SwiftUI + AppKit
- 窗口：NSPanel / NSWindow
- 动画：SwiftUI Animation、Core Animation，必要时使用 Metal 或 Canvas
- 数据：UserDefaults、SwiftData 或 SQLite、Keychain

## 核心形态

启动后，应用在主屏幕顶部正中间显示胶囊。

收起态建议尺寸：

```text
宽度：180 - 240 px
高度：38 - 46 px
顶部间距：8 - 14 px
圆角：高度的一半
```

展开态建议尺寸：

```text
宽度：420 - 620 px
高度：160 - 320 px
圆角：28 - 36 px
```

特殊模块可以使用更大展开态，例如 AI 对话、剪贴板历史、任务队列。

## 视觉要求

- 无系统标题栏、边框、关闭按钮
- 背景透明，主体为深色半透明或毛玻璃质感
- 支持浅色模式和深色模式
- 胶囊边缘有柔和高光
- 阴影轻，不做厚重卡片感
- Retina 屏幕下边缘清晰
- 默认收起态有低干扰流动动画
- Hover 时亮度、高光和尺寸有轻微反馈
- 降低动态效果开启时，动画自动减少

## 动画要求

所有状态切换必须连续，不使用突兀的隐藏/显示。

动画包括：

- 胶囊展开为面板
- 面板收起为胶囊
- Hover 反馈
- 点击按压反馈
- 模块切换
- 通知进入
- 任务进度变化
- 屏幕变化后的重新定位

推荐时长：

```text
点击反馈：80 - 140 ms
展开动画：320 - 520 ms
收起动画：240 - 420 ms
模块切换：180 - 300 ms
通知插入：260 - 420 ms
```

展开时，宽度、高度、圆角、阴影、背景材质、内容透明度必须同步变化。

## 窗口行为

- 顶部居中显示
- 支持所有 Space
- 支持全屏应用辅助显示
- 默认不抢焦点
- 点击展开后，输入类模块可以获取焦点
- 点击外部自动收起
- Esc 收起
- 支持拖拽调整位置
- 支持恢复默认位置
- 支持菜单栏管理

多屏规则：

- 默认显示在主屏幕顶部正中间
- 可选择固定到指定屏幕
- 可选择跟随鼠标所在屏幕
- 分辨率、缩放、屏幕插拔变化后自动重新定位
- 避开菜单栏、刘海和圆角屏幕安全区域

## 状态优先级

```text
用户主动展开 > 紧急提醒 > 正在进行的任务 > 媒体播放 > 日程提醒 > 系统状态 > 默认待机
```

当多个状态同时存在：

- 收起态只显示最高优先级信息
- 展开态可以显示多个模块
- 用户主动操作优先级最高
- 通知有展示时长
- 媒体播放停止后延迟回到默认状态

## 权限

权限按需申请，不在首次启动一次性请求。

- 辅助功能：全局快捷键、部分系统控制
- 屏幕录制：识别窗口内容或截图
- 自动化：控制系统设置或播放器
- 日历：读取日程
- 提醒事项：读取待办
- 通知：发送提醒
- 剪贴板：读取历史
- 网络：AI 服务

## 隐私

- 剪贴板历史可关闭
- 敏感内容默认不上传
- AI 功能发送内容前明确提示
- 提供一键清除本地数据
- 提供隐私模式
- Keychain 存储 API Key 和 token

## 性能目标

```text
待机 CPU：接近 0%
普通动画 CPU：低于 3%
内存：尽量低于 150 MB
动画帧率：60 FPS
高刷屏：尽量适配 120 Hz
```

实现要求：

- 不可见或静止状态暂停动画
- 降低动态效果时减少绘制
- 避免高频轮询
- 系统状态优先事件驱动
- 后台任务按模块调度

## 验收标准

- 启动后稳定显示在屏幕顶部正中间
- 胶囊无系统窗口边框
- 胶囊有流畅且克制的流动动画
- 点击展开动画自然、连续、优雅
- 多模块切换无闪烁、无跳变
- 媒体播放模块可降级运行
- 设置中心完整
- 多屏、全屏、睡眠唤醒后位置正确
- 权限拒绝时有清晰降级逻辑
- 长时间运行稳定
- 隐私数据可控、可清除
